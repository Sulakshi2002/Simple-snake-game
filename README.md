# Snake Game Using Python Turtle

## Overview
A classic Snake game implemented in Python using the Turtle graphics library. Players control a snake to collect apples, grow in length, and avoid collisions with walls or itself. The game tracks the current score and high score dynamically.

## Features
- Arrow key controls for snake movement
- Randomly spawning apples for the snake to eat
- Snake grows longer with each apple collected
- Collision detection with walls and self
- Score and high score display
- Adjustable game speed for increasing difficulty

## Requirements
- Python 3.x
- Built-in `turtle`, `time`, and `random` modules (no extra installation needed)

## How to Run
1. Clone or download the project.
2. Run the script using Python:
